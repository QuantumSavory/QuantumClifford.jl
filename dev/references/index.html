Suggested reading

For the basis of the tableaux methods first read Daniel Gottesman (1998) followed by the more efficient approach described in Scott Aaronson, Daniel Gottesman (2004).

The tableaux can be canonicalized (i.e. Gaussian elimination can be performed on them) in a number of different ways, and considering the different approaches provides useful insight. The following methods are implemented in this library:

  • The default one: Hector J Garcia, Igor L Markov, Andrew W Cross (2012)
  • Useful when in need of tracing out a set of qubits: Koenraad MR Audenaert, Martin B Plenio (2005)
  • Useful when defining logical operators of codes: Daniel Gottesman (1997)

For the use of these methods in error correction and the subtle overlap between the two fields consider these resources. They are also useful in defining some of the specific constraints in commutation between rows in the tableaux:

  • Andrew M Steane (2007)
  • A Robert Calderbank, Eric M Rains, PM Shor, Neil JA Sloane (1998)
  • David JC MacKay, Graeme Mitchison, Paul L McFadden (2004)
  • Mark M Wilde (2009)

These publications describe the uniform sampling of random stabilizer states:

  • Robert Koenig, John A Smolin (2014)
  • Sergey Bravyi, Dmitri Maslov (2021)
  • Ewout Van Den Berg (2021)
  • Yaodong Li, Xiao Chen, Matthew PA Fisher (2019)
